# Webpack-start-template

## Commands:

### Deployment to Github-Pages

```shell
pnpm run gh-pages
```

### Starting a server for development

```shell
pnpm run dev-server
```

### Building a project without optimization

```shell
pnpm run build-dev
```

### Build the project with optimization

```shell
pnpm run build-prod
```

### Cleaning the dist folder

```shell
pnpm run clear
```
